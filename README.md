# ModernJS Mastery Exercise: Marvel Cards
## Description
> Milestone challenge to dynamically create cards of superheroes using promises to load JSON data and functions to write cards based on team, gender, and styling.

* Jquery
* NPM: Grunt
* Bootstrap
